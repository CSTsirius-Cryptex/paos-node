⚠️ 請先安裝 Python：

  1. 前往 Python 官方下載頁面
  2. 下載 Python 3.10 以上版本（Windows installer）
  3. 安裝時務必勾選「Add Python to PATH」
  4. 安裝完成後點「重新檢查」

⚠️ 請先安裝 cloudflared：

  1. 前往 Cloudflare 官方下載頁面
  2. 下載 Windows 64-bit 版本（cloudflared-windows-amd64.exe）
  3. 將檔案改名為 cloudflared.exe，放到 C:\Windows\System32\
  4. 安裝完成後點「重新檢查」
